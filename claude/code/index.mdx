Claude Code est un outil de développement assisté par IA qui s'installe dans votre terminal, lit votre code source, exécute des commandes et interagit avec vos outils de développement — le tout en langage naturel.

À la différence d'un simple générateur de code, Claude Code est un **agent** : il ne se contente pas de répondre, il agit. Il lit vos fichiers, comprend l'architecture de votre projet, modifie du code sur plusieurs fichiers simultanément, lance des tests, et corrige les erreurs qu'il détecte — de façon autonome ou en vous demandant une validation à chaque étape.

## Ce que vous allez apprendre

Ce module couvre Claude Code de façon progressive, du premier lancement jusqu'aux usages avancés :

- **Comprendre** comment fonctionnent les assistants de codage et pourquoi Claude excelle dans l'utilisation des outils
- **Installer et configurer** Claude Code dans votre environnement
- **Gérer le contexte** de votre projet avec le fichier `CLAUDE.md`
- **Apporter des modifications** à votre code efficacement, y compris via des captures d'écran
- **Contrôler la conversation** pour rester concentré sur vos objectifs
- **Créer des commandes personnalisées** pour automatiser vos workflows répétitifs
- **Étendre Claude Code** avec des serveurs MCP comme Playwright
- **Intégrer GitHub** pour automatiser les revues de pull requests
- **Utiliser les hooks** pour contrôler précisément ce que Claude peut lire et modifier
- **Exploiter le SDK** pour intégrer Claude Code dans vos propres pipelines

## Prérequis

- Node.js installé sur votre machine ([instructions](https://nodejs.org/en/download))
- Un compte Anthropic avec accès à l'API ou un abonnement Claude Pro/Max
- Une connaissance basique du terminal

<Tip>
Claude Code fonctionne avec n'importe quel projet, quel que soit le langage ou le framework. Vous n'avez pas besoin d'un projet spécifique pour suivre ce module — votre code existant convient parfaitement.
</Tip>
